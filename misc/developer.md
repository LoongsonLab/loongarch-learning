# 龙芯爱好者社区

龙芯爱好者社区是一个由第三方爱好者、行业人员与学生组成的，致力于龙芯（龙架构）软硬件生态建设的互联网社区。

其包含的内容有：
- 新手资源。包含有踩坑指南、开发指南等入门教程。
- 数据材料。包含有芯片规格、产品规格、软硬件兼容数据等信息。
- 社区资源。包含有交流论坛、技术分享等内容。

社区网址链接：https://loongfans.cn/

# 龙芯开源社区

龙芯开源社区是龙芯处理器的基础软件生态建设平台，包含有：
- Loongnix系统安装镜像、虚拟机镜像等下载。
- 龙芯浏览器。
- 提供 LoongArch 架构下最新 JDK。
- 提供 LoongArch 架构下最新 .net。
- 提供 LoongArch 架构下的 FFmpeg 与 X264。
- Electron 框架支持。
- 以及其他基础软件、图形视频驱动、调试工具、应用工具等。

社区网址链接：https://www.loongnix.cn/zh/

# 龙芯应用合作社

龙芯应用合作社，提供龙架构下适配的系统软件，方便下载、安装和使用。

网址链接：https://app.loongapps.cn/home

# loongson生态创新中心

龙芯生态创新衷心，包含有：
- helm LoongArch 源
- MongoDB 支持 LoongArch 架构版
- openstack 支持 LoongArch 架构版

以及其他前沿创新应用。

网址链接：https://gitee.com/cn-loongson
