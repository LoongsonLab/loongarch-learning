# 其他的参考文档

## **龙架构与操作系统底层的交互**

此文档主要是介绍龙架构底层如何与操作系统内核配合。详细的介绍了龙架构一些重点内容，结合代码例子。

可以说是对龙架构官方手册的进一步解释与说明。

目的： **读完这个文档，你自己可以很快的给一个操作系统内核增加LoongArch的支持。**

内容整体如下： 

- 1. 本文档是对LoongArch架构与底层操作系统的紧密相关文档，类似于对linux内核的目录arch/loongarch下做个说明。
- 2. 特权态指令的一些说明，结合模拟器做出伪代码的说明。 
- 3. DMW的使用与说明，以及使用的场合说明。 
- 4. LA的内存管理系统，包括TLB相关寄存器的初始化，LA的TLB refill指令，tlb的使用等等（这部分也是难点） 
- 5. LA的中断系统，如何更加采用一个合理的中断安排等 
- 6. LoongArch的新世界与旧世界，我们为什么要采用新世界，如何使用新世界等等 
- 7. LoongArch ABI2.0的介绍与使用，伪代码实例等等
- 8. 如何调试一个内核，以一个内核为例，如何使用上游的qemu以及gdb支持调试操作系统等等。

文档在[这里](https://os-kernel-with-loong64-doc.readthedocs.io/en/latest/)
